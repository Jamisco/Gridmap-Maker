Detailed Description

A layered mesh is a collection of fused meshes that are then combined together to make one mesh. Each fused mesh is unique visually, meaning it may have its own different texture, color, etc. However each fused mesh has thesame Shape. 5 meshes with 5 materials means 5 game objects to update, 5 renderer components 1 mesh with 5 materials means 1 game object to update, 1 renderer component.

The benefit of 5 meshes is that each part can be separately culled if necessary, potentially reducing draw calls. The benefit of 1 mesh is less CPU and GPU overhead.
